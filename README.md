# Ubuntu

1. Start docker -

   sudo service docker start

2. Running CentOS

Get Centos Container docker run -d centos tail -f /dev/null

To open BAsh :
docker exec -it sweet_franklin bash

To insatll Java :
yum install java

To Run the Docker File on local system docker build -t spring-boot-docker .
